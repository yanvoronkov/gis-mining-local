# Установка Bitrix24 (для разработки)

## Предварительные требования
Убедитесь, что у вас установлен **Docker Desktop** для Mac (Apple Silicon).
Если нет, скачайте его здесь: [https://www.docker.com/products/docker-desktop/](https://www.docker.com/products/docker-desktop/)

## Запуск
1. Откройте терминал в этой папке.
2. Запустите окружение командой:
   ```bash
   docker compose up -d --build
   ```
3. Дождитесь завершения сборки и запуска контейнеров.

## Установка Битрикса
1. Откройте браузер и перейдите по адресу: [http://localhost/bitrixsetup.php](http://localhost/bitrixsetup.php)
2. Следуйте инструкциям установщика.
   - Выберите дистрибутив (например, "Управление сайтом" или "Корпоративный портал" / "Демо-версия").
   - На этапе настройки базы данных укажите:
     - **Сервер:** `db`
     - **Пользователь:** `bitrix`
     - **Пароль:** `bitrix`
     - **Имя базы данных:** `bitrix`

## Остановка
Чтобы остановить сервер, выполните:
```bash
docker compose down
```
